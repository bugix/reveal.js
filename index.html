Git [ɡɪt] ist eine freie Software zur verteilten Versionsverwaltung von Dateien, die durch Linus Torvalds initiiert wurde.

Netzbasierte Dienste zur Versionsverwaltung für Software-Entwicklungsprojekte.

Eine Versionsverwaltung ist verwendet wird. Alle Versionen werden in einem Archiv mit Zeitstempel und Benutzerkennung gesichert und können später wiederhergestellt werden.
